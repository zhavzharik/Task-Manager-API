# Task-Manager-API

# Функционал

 ⁃ Создавать задачи
 
 ⁃ Редактировать задачи
 
 ⁃ Удалять задачи
 
 ⁃ Помечать задачу выполненной
 

При создании задачи указывается ее тип (рабочая или личная), дату завершения задачи и описание.

# Стек

 ⁃ Asp.Net.Core 5
 
 ⁃ EntityFramework.Core 5 (хранение в памяти)
 
 ⁃ REST.API
 
 ⁃ Swagger (Swashbuckle)
